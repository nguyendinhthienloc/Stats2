\textit{%
This report investigates regularised regression methods for predicting body-fat
percentage (brozek) from anthropometric measurements in the \texttt{fat.csv}
dataset ($n = 252$, $p = 14$ candidate predictors).  After excluding
leakage-prone variables, we compare Ordinary Least Squares (OLS), Ridge, Lasso,
and Elastic Net regression using 5-fold cross-validation.  We further explore
fixed random ReLU feature expansions to bridge classical penalisation with
neural-network concepts.
