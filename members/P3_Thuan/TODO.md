# Nhiệm vụ của Lê Minh Thuận (P3)

Chào Thuận, đây là hướng dẫn công việc của bạn:

**Nhánh làm việc (Branch):**
Hãy chắc chắn bạn đang ở nhánh của mình trước khi viết code:
`git checkout feature/p3-thuan-lasso-compare`

**Nhiệm vụ chính:** Mô hình Lasso & So sánh các mô hình

**Bạn cần chỉnh sửa các file sau:**
1. Code R: `R_models/02_lasso.R` và `R_models/02_comparison.R`
2. Báo cáo LaTeX: `report/sections/02_ols_ridge_lasso.tex` (Phần 2.3, 2.4 và 2.5)

**Mục tiêu (Phase 2):**
- Đợi Lộc (P1) hoàn thành dữ liệu chuẩn.
- Code mô hình Lasso, vẽ đồ thị CV và shrinkage.
- Đợi Tuấn (P2) code xong OLS và Ridge. Sau đó, thu thập kết quả và chạy `02_comparison.R` để so sánh OLS, Ridge, Lasso.
- Viết báo cáo giải thích mô hình Lasso, so sánh các mô hình và **chọn ra mô hình tốt nhất (Core Model)** để thử nghiệm.
